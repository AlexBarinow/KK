1) Написать программу, которая будет просить у пользователя ввести одномерный массив, а программа его записывается зеркально (т.е. пользователь вводит первый элемент массива, а программа записывает его как конечный элемент, и т.д.). Вывести этот зеркальный массив на экран.
2) Написать программу, которая будет просить у пользователя ввести одномерный массив целочисленных значений, требуется вывести сумму этих элементов.
3) Написать программу, которая будет просить у пользователя ввести одномерный массив типа String, вывести на экран элементы, если в них будет содержаться слово «программист».
4) Написать программу, которая будет просить у пользователя ввести двумерный массив (матрицу), необходимо её транспонировать и вывести на экран.
5) Написать программу, которая будет просить у пользователя ввести два двумерных массив (две матрицы), сложить две матрицы, вывести на экран.
6) Написать программу, которая будет просить у пользователя ввести два двумерных массив (две матрицы), перемножить матрицы, вывести на экран.
7) Написать программу, которая будет просить у пользователя ввести двумерный массив (матрицу), проверить её на ортогональность, вывести ответ пользователю.
8) Написать программу, которая будет заполнять нулями матрицу так, чтобы она стала треугольной. Вывести её на экран.
9) Финальная задача. Задача со звёздочкой. Попробовать догадаться, как можно вывести на экран все матрицы (двумерные массивы) с помощью цикла foreach.
